Command line callers



 normal 

>> npx playwright test // by default is headless with defalt browser


 visual mode (headed) -> non headless 

>> npx playwright test --headed // visual mode


 specifygin browser types 

>> npx playwright test --browser=firefox // [chrome|firefox|webkit]
>> npx playwright test --browser=all // [chrome|firefox|webkit]


 browser with visual mode (headed) 

>> npx playwright test --headed --browser=webkit


 with specific path of the test 

>> npx playwright test tests/example.spec.ts



 Running tagged tests - "Some description @myTag" 

>> npx playwright test --grep @myTag



Calling with specific config file

>> npx playwright test --config=playwright.config.ts --project=Firefox



Using reporters

>> npx playwright test --config=playwright.config.ts --project=Chromium --reporter=line
>> npx playwright test --config=playwright.config.ts --project=Chromium --reporter=list (by default)
>> npx playwright test --config=playwright.config.ts --project=Chromium --reporter=dot
>> npx playwright test --config=playwright.config.ts --project=Chromium --reporter=junit
>> npx playwright test --config=playwright.config.ts --project=Chromium --reporter=html
  --> npx playwright show-report // shows previously builded report
